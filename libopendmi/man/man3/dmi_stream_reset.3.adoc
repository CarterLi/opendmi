== Name

dmi_stream_reset - reset the stream cursor to the beginning.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

void dmi_stream_reset(dmi_stream_t *stream);
----

== Description

The `dmi_stream_reset` function repositions the stream cursor back to the
start of the entity body so that it can be read again from the beginning.

=== Parameters

`stream`::
Stream to reset.
