== Name

dmi_entity_data - get pointer to raw SMBIOS structure data.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/entity.h>

const void *dmi_entity_data(const dmi_entity_t *entity, dmi_type_t type);
----

== Description

The `dmi_entity_data` function returns a pointer to the raw SMBIOS structure
data, including the structure header. The entity type is verified against the
expected `type` parameter before returning the data.

=== Parameters

`entity`::
Entity descriptor.

`type`::
Expected SMBIOS structure type. If the entity's type does not match, the
function returns `NULL` and raises an error. Pass `DMI_TYPE_INVALID` to skip
type checking.

== Return value

Returns a pointer to the raw SMBIOS structure data. The returned pointer must
not be modified or freed by the caller. Returns `NULL` if `entity` is `NULL`
or the entity type does not match `type`.

== Errors

If the entity type does not match `type`, a `DMI_ERROR_INVALID_ENTITY_TYPE`
error is raised on the entity's context.
