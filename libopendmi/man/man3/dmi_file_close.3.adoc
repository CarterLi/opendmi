== Name

dmi_file_close - close an open file descriptor.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/utils/file.h>

int dmi_file_close(int fd);
----

== Description

The `dmi_file_close` function is a cross-platform wrapper around `close`(2).

=== Parameters

`fd`::
Open file descriptor to close.

== Return value

Returns `0` on success, or `-1` on error with `errno` set accordingly.
