== Name

dmi_stream_read_data_at - read data from the stream at a given offset.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

bool dmi_stream_read_data_at(
        const dmi_stream_t *stream,
        void               *ptr,
        size_t              offset,
        size_t              length);
----

== Description

The `dmi_stream_read_data_at` function copies `length` bytes starting at
`offset` from the beginning of the entity body into the buffer pointed to by
`ptr`. The stream cursor is not modified. No byte-order conversion is
performed.

=== Parameters

`stream`::
Stream to read from.

`ptr`::
Buffer that receives the data; must be at least `length` bytes.

`offset`::
Byte offset from the beginning of the entity body.

`length`::
Number of bytes to read.

== Return value

Returns `true` on success, `false` if the requested range exceeds the stream
bounds.
