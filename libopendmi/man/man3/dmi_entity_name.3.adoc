== Name

dmi_entity_name - get entity type name.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/entity.h>

const char *dmi_entity_name(const dmi_entity_t *entity);
----

== Description

The `dmi_entity_name` function returns the human-readable name of the entity's
SMBIOS structure type (e.g., "BIOS Information", "System Information",
"Cooling Device").

=== Parameters

`entity`::
Entity descriptor.

== Return value

Returns a pointer to a string containing the type name. The returned string
must not be modified or freed by the caller. If `entity` is `NULL`, returns
`NULL`.
