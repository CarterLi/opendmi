== Name

dmi_error_clear - remove all error descriptors from the error queue.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/error.h>

void dmi_error_clear(dmi_context_t *context);
----

== Description

The `dmi_error_clear` function removes all error descriptors from the error
queue of `context`, freeing any associated message strings. After the call the
queue is empty. If `context` is `NULL`, the function does nothing.

=== Parameters

`context`::
DMI context.
