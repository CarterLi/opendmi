== Name

dmi_entity_handle - get entity handle.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/entity.h>

dmi_handle_t dmi_entity_handle(const dmi_entity_t *entity);
----

== Description

The `dmi_entity_handle` function returns the SMBIOS handle associated with the
given entity. Each SMBIOS structure is assigned a unique handle by the platform
firmware, which can be used to reference the structure from other structures.

=== Parameters

`entity`::
Entity descriptor.

== Return value

Returns the handle associated with the entity. If `entity` is `NULL`, returns
`DMI_HANDLE_INVALID` (`0xFFFF`).
