== Name

dmi_stream_initialize - initialize a stream for reading an entity's body data.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

bool dmi_stream_initialize(dmi_stream_t *stream, const dmi_entity_t *entity);
----

== Description

The `dmi_stream_initialize` function sets up the stream to read the body of
`entity`, positioning the cursor at the beginning of the body (past the SMBIOS
structure header).

=== Parameters

`stream`::
Stream to initialize.

`entity`::
Entity whose body data will be read.

== Return value

Returns `true` on success, `false` if `stream` or `entity` is `NULL`.
