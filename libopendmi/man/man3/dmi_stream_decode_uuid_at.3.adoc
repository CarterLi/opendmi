== Name

dmi_stream_decode_uuid_at - read and decode a UUID from the stream at a given offset.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode_uuid_at(stream, offset, pvalue)
----

== Description

The `dmi_stream_decode_uuid_at` macro reads 16 bytes at `offset` from the
beginning of the entity body, converts the UUID from SMBIOS byte order into
RFC 4122 representation using `dmi_uuid_decode`, and stores the result in the
variable pointed to by `pvalue`. The stream cursor is not modified.

=== Parameters

`stream`::
Stream to read from.

`offset`::
Byte offset from the beginning of the entity body.

`pvalue`::
Pointer to a variable that receives the decoded UUID.

== Return value

Returns `true` on success, `false` if the requested range exceeds the stream
bounds.
