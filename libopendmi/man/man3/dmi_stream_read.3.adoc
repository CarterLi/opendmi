== Name

dmi_stream_read - read a typed value from the stream at the current position.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_read(stream, pvalue)
----

== Description

The `dmi_stream_read` macro reads `sizeof(*pvalue)` bytes from the stream into
the variable pointed to by `pvalue` and advances the stream cursor. No
byte-order conversion is performed. This is a convenience wrapper around
`dmi_stream_read_data`(3).

=== Parameters

`stream`::
Stream to read from.

`pvalue`::
Pointer to the variable that receives the raw data.

== Return value

Returns `true` on success, `false` if there are not enough bytes remaining.
