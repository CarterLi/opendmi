== Name

dmi_error_get_first - remove and return the oldest error descriptor from the error queue.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/error.h>

dmi_error_t *dmi_error_get_first(dmi_context_t *context);
----

== Description

The `dmi_error_get_first` function removes the oldest error descriptor from
the error queue of `context` and returns a pointer to it. It can be called
repeatedly until there are no more entries.

The returned pointer remains valid until the error queue is next modified by
`dmi_error_raise`(3), `dmi_error_raise_ex`(3), `dmi_error_get_first`(3),
`dmi_error_get_last`(3), or `dmi_error_clear`(3).

=== Parameters

`context`::
DMI context.

== Return value

Returns a pointer to the oldest `dmi_error_t` removed from the queue, or
`NULL` if `context` is `NULL` or the error queue is empty.

== Examples

[source, c]
```
dmi_error_t *error;

while ((error = dmi_error_get_first(context)) != NULL) {
    fprintf(stderr, "%s:%d: %s\n",
            error->function,
            error->line,
            dmi_error_message(error->reason));
}
```
