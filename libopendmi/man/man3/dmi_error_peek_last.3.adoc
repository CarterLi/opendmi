== Name

dmi_error_peek_last - return the newest error descriptor from the error queue without removing it.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/error.h>

dmi_error_t *dmi_error_peek_last(dmi_context_t *context);
----

== Description

The `dmi_error_peek_last` function returns a pointer to the newest error
descriptor in the error queue of `context` without removing it. The queue is
not modified.

=== Parameters

`context`::
DMI context.

== Return value

Returns a pointer to the newest `dmi_error_t` in the queue, or `NULL` if
`context` is `NULL` or the error queue is empty.
