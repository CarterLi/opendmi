== Name

dmi_stream_decode_str - read and decode an SMBIOS string reference from the stream at the current position.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode_str(stream, pvalue)
----

== Description

The `dmi_stream_decode_str` macro reads a `dmi_string_t` index from the stream
and resolves it to a string pointer via `dmi_entity_string`. The stream cursor
is advanced.

=== Parameters

`stream`::
Stream to read from.

`pvalue`::
Pointer to a variable that receives the resolved string.

== Return value

Returns `true` on success, `false` if there are not enough bytes remaining.
