== Name

dmi_stream_decode_bcd_at - read and decode a BCD-encoded value from the stream at a given offset.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode_bcd_at(stream, offset, type, pvalue)
----

== Description

The `dmi_stream_decode_bcd_at` macro reads `sizeof(type)` bytes at `offset`
from the beginning of the entity body, decodes the value from BCD encoding
using `dmi_decode_bcd`, and stores the result in the variable pointed to by
`pvalue`. The stream cursor is not modified.

=== Parameters

`stream`::
Stream to read from.

`offset`::
Byte offset from the beginning of the entity body.

`type`::
Wire-format type to read (e.g., `dmi_byte_t`).

`pvalue`::
Pointer to the variable that receives the decoded value.

== Return value

Returns `true` on success, `false` if the requested range exceeds the stream
bounds.
