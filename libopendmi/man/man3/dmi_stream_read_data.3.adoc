== Name

dmi_stream_read_data - read data from the stream at the current position.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

bool dmi_stream_read_data(dmi_stream_t *stream, void *ptr, size_t length);
----

== Description

The `dmi_stream_read_data` function copies `length` bytes from the stream into
the buffer pointed to by `ptr` and advances the stream cursor. No byte-order
conversion is performed.

=== Parameters

`stream`::
Stream to read from.

`ptr`::
Buffer that receives the data; must be at least `length` bytes.

`length`::
Number of bytes to read.

== Return value

Returns `true` on success, `false` if there are fewer than `length` bytes
remaining in the stream.
