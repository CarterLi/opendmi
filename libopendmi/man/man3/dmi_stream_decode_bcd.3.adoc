== Name

dmi_stream_decode_bcd - read and decode a BCD-encoded value from the stream at the current position.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode_bcd(stream, type, pvalue)
----

== Description

The `dmi_stream_decode_bcd` macro reads `sizeof(type)` bytes from the stream,
decodes the value from BCD encoding using `dmi_decode_bcd`, and stores the
result in the variable pointed to by `pvalue`. The stream cursor is advanced.

=== Parameters

`stream`::
Stream to read from.

`type`::
Wire-format type to read (e.g., `dmi_byte_t`).

`pvalue`::
Pointer to the variable that receives the decoded value.

== Return value

Returns `true` on success, `false` if there are not enough bytes remaining.
