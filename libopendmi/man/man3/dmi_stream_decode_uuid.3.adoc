== Name

dmi_stream_decode_uuid - read and decode a UUID from the stream at the current position.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode_uuid(stream, pvalue)
----

== Description

The `dmi_stream_decode_uuid` macro reads 16 bytes from the stream, converts
the UUID from SMBIOS byte order into RFC 4122 representation using
`dmi_uuid_decode`, and stores the result in the variable pointed to by
`pvalue`. The stream cursor is advanced.

=== Parameters

`stream`::
Stream to read from.

`pvalue`::
Pointer to a variable that receives the decoded UUID.

== Return value

Returns `true` on success, `false` if there are not enough bytes remaining.
