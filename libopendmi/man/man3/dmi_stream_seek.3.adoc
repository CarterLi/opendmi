== Name

dmi_stream_seek - seek to a byte position within the stream.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

bool dmi_stream_seek(dmi_stream_t *stream, size_t position);
----

== Description

The `dmi_stream_seek` function repositions the stream cursor to `position`
bytes from the beginning of the entity body.

=== Parameters

`stream`::
Stream to reposition.

`position`::
Byte offset from the beginning of the entity body.

== Return value

Returns `true` on success, `false` if `position` is beyond the end of the
stream.
