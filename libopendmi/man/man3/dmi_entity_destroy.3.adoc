== Name

dmi_entity_destroy - destroy entity descriptor.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/entity.h>

void dmi_entity_destroy(dmi_entity_t *entity);
----

== Description

The `dmi_entity_destroy` function releases all resources associated with the
given entity descriptor, including the decoded structure data, the string
table, and the entity descriptor itself.

If `entity` is `NULL`, this function does nothing.

If a type-specific cleanup handler is registered and decoded data is present,
the cleanup handler is invoked before freeing the decoded data.

=== Parameters

`entity`::
Entity descriptor to destroy, or `NULL`.

== Return value

None.
