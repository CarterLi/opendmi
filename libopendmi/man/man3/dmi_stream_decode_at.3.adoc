== Name

dmi_stream_decode_at - read and decode a value from the stream at a given offset.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode_at(stream, offset, type, pvalue)
----

== Description

The `dmi_stream_decode_at` macro reads `sizeof(type)` bytes at `offset` from
the beginning of the entity body, converts the value from SMBIOS wire format
(little-endian) into host byte order using `dmi_decode`, and stores the result
in the variable pointed to by `pvalue`. The stream cursor is not modified.

=== Parameters

`stream`::
Stream to read from.

`offset`::
Byte offset from the beginning of the entity body.

`type`::
Wire-format type to read (e.g., `dmi_word_t`, `dmi_dword_t`).

`pvalue`::
Pointer to the variable that receives the decoded value.

== Return value

Returns `true` on success, `false` if the requested range exceeds the stream
bounds.
