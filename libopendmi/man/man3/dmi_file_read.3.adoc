== Name

dmi_file_read - read exactly the requested number of bytes from a file descriptor.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/utils/file.h>

ssize_t dmi_file_read(int fd, dmi_data_t *data, size_t size);
----

== Description

The `dmi_file_read` function performs a full read by looping until all
requested bytes are read or end-of-file is reached. It retries automatically
if interrupted by a signal (`EINTR`). Fewer than `size` bytes are returned
only if EOF is encountered before the read is complete.

=== Parameters

`fd`::
Open file descriptor to read from.

`data`::
Buffer that receives the data; must be at least `size` bytes.

`size`::
Number of bytes to read.

== Return value

Returns the total number of bytes read (may be less than `size` at EOF), or
`-1` on error with `errno` set accordingly.
