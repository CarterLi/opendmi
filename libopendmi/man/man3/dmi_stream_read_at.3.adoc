== Name

dmi_stream_read_at - read a typed value from the stream at a given offset.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_read_at(stream, offset, pvalue)
----

== Description

The `dmi_stream_read_at` macro reads `sizeof(*pvalue)` bytes at `offset` from
the beginning of the entity body into the variable pointed to by `pvalue`. The
stream cursor is not modified. No byte-order conversion is performed. This is
a convenience wrapper around `dmi_stream_read_data_at`(3).

=== Parameters

`stream`::
Stream to read from.

`offset`::
Byte offset from the beginning of the entity body.

`pvalue`::
Pointer to the variable that receives the raw data.

== Return value

Returns `true` on success, `false` if the requested range exceeds the stream
bounds.
