== Name

dmi_stream_is_done - check whether all data in the stream has been consumed.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

bool dmi_stream_is_done(const dmi_stream_t *stream);
----

== Description

The `dmi_stream_is_done` function checks whether the stream cursor has reached
the end of the entity body, meaning there are no remaining bytes to read.

=== Parameters

`stream`::
Stream to check.

== Return value

Returns `true` if there are no remaining bytes to read, `false` otherwise.
