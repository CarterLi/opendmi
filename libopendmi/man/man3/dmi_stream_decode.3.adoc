== Name

dmi_stream_decode - read and decode a value from the stream at the current position.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/stream.h>

dmi_stream_decode(stream, type, pvalue)
----

== Description

The `dmi_stream_decode` macro reads `sizeof(type)` bytes from the stream,
converts the value from SMBIOS wire format (little-endian) into host byte
order using `dmi_decode`, and stores the result in the variable pointed to by
`pvalue`. The stream cursor is advanced.

=== Parameters

`stream`::
Stream to read from.

`type`::
Wire-format type to read (e.g., `dmi_word_t`, `dmi_dword_t`).

`pvalue`::
Pointer to the variable that receives the decoded value.

== Return value

Returns `true` on success, `false` if there are not enough bytes remaining.
