== Name

dmi_entity_type - get entity SMBIOS type.

== Library

OpenDMI C library (`libopendmi`, `-lopendmi`)

== Synopsis

[source, c]
----
#include <opendmi/entity.h>

dmi_type_t dmi_entity_type(const dmi_entity_t *entity);
----

== Description

The `dmi_entity_type` function returns the SMBIOS structure type of the given
entity. SMBIOS types 0 through 127 are defined by the SMBIOS specification;
types 128 through 255 are available for OEM- and system-specific information.

=== Parameters

`entity`::
Entity descriptor.

== Return value

Returns the SMBIOS type of the entity. If `entity` is `NULL`, returns
`DMI_TYPE_INVALID`.
